# Agentic AI Fraud Detection System

This project builds an advanced fraud detection system using machine learning, time-series analysis, and agent-based AI reasoning.

## Dataset

The dataset used in this project is the Credit Card Fraud Detection dataset from Kaggle:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Note: The dataset is highly imbalanced, with fraudulent transactions representing approximately 0.17% of all transactions.

## Features
- Data cleaning and preprocessing
- Feature engineering (time + location simulation)
- Model comparison (Logistic Regression, Decision Tree, Random Forest)
- Time-series analysis
- Agentic AI system
- RAG-based verification

## Results
- ROC-AUC: ~0.95
- Precision: ~96%
- Strong fraud detection performance
